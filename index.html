A Propos

La plateforme


Notre ambition est de construire et de mettre à disposition du plus grand nombre un écosystème complet et Open source autour des Offres d'emploi en Nouvelle-Calédonie. Le logiciel est open source. Nous espérons attirer ds utilisateurs finaux, des développeurs, des intégrateurs, des étudiants stagiaires qui souhaitent apprendre et progresser tout en participant à un produit utile au plus grand nombre, pour nous, les calédoniens.


  • Sdk
  • API
  • Application Mobile

Développement d'un SDK Java qui permet d'interagir simplement avec le site des Offres d'Emploi du Gouvernement de Nouvelle-Calédonie.

Le SDK est disponible via une dépendance : .

Pour contribuer rendez-vous sur : https://github.com/adriens/emploi-nc-sdk/issues.

L'API utilise notre SDK.

Développé une API REST (https://github.com/adriens/emploi-nc-api) qui utilise le SDK.

Docker: nous avons prévu de rendre l'API disponible sur Docker.

Déployé sur Heroku et mis à disposition cette API sur le Marketplace d'APIS.

Pour en savoir plus : https://rapidapi.com/adriens/api/emploi-nouvelle-caledonie.
