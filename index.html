Notre Démarche


Nous développons un écosystème complet et Open source autour des Offres d'emploi en Nouvelle-Calédonie. Le but est de mettre en oeuvre, gratuitement, et avec du logiciel Open Source des services packagés comme le font les géants du Web et d'encourager l'Open Innovation en Nouvelle-Calédonie.


  • Sdk
  • API
  • Application Mobile

Développement d'un SDK Java qui permet d'interagir simplement avec le site des Offres d'Emploi du Gouvernement de Nouvelle-Calédonie

Le SDK est disponible via une dépendance : (https://jitpack.io/#adriens/emploi-nc-sdk)

L'API utilise notre SDK.

Développé une API REST (https://github.com/adriens/emploi-nc-api) qui utilise le SDK.

Docker: nous avons prévu de rendre l'API disponible sur Docker.

Déployé sur Heroku et mis à disposition cette API sur le Marketplace d'APIS (https://rapidapi.com/adriens/api/emploi-nouvelle-caledonie).
